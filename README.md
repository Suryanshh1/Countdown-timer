# Countdown Timer Project

A modern animated countdown timer built using HTML, CSS, and JavaScript.

## Features
- Date & time selection
- Optional event name
- Real-time countdown
- Progress bar
- Confetti animation on completion
- Sound alert
- Browser notification

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## How to Run
1. Download all files
2. Open `index.html` in any browser
3. Select date & time
4. Click **Start Timer**
